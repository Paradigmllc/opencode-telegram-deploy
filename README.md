# opencode-telegram-deploy

Deploy artifacts for the Paradigm OpenCode Telegram Bot — designed to be deployed
on Coolify as a Docker Compose application. Source of truth lives in
`Paradigmllc/dotfiles` (private) under `opencode-telegram/`.

This repo is intentionally public so Coolify can `git clone` it without auth.
**No secrets are committed here** — all credentials come from Coolify env vars.
